1️⃣  TELEGRAM CREDENTIALS (5 min)
    
    a) Go to: https://my.telegram.org
    b) Login with your Telegram account
    c) API Development Tools → Create Application
    d) Note: api_id and api_hash
    e) Edit .env.local:
       
       TELEGRAM_API_ID=YOUR_ID_HERE
       TELEGRAM_API_HASH=YOUR_HASH_HERE
       TELEGRAM_PHONE=+YOUR_PHONE_WITH_CODE

2️⃣  TEST CHANNEL SETUP (3 min)
    
    a) Open Telegram
    b) Create New Channel → @gramgpt_e2e_test
    c) Settings → Discussion → "Create group"
    d) Add yourself as Admin (with "Edit messages")
    e) Update .env.local:
       
       TEST_CHANNEL=gramgpt_e2e_test

3️⃣  AUTHORIZE TELEGRAM (2 min)
    
    Run in new terminal:
    
    python -c "
    from src.services.telegram_user_client import TelegramUserClient
    client = TelegramUserClient(
        api_id=YOUR_API_ID,
        api_hash='YOUR_API_HASH',
        phone='+YOUR_PHONE'
    )
    client.connect()  # Will ask for 2FA code
    print('✅ Authorized!')
    "
    
    (Telegram will send you a code)

4️⃣  RUN FIRST CAMPAIGN (2 min)

      -H "Content-Type: application/json" \
      -d '{"name": "First Test", "goal": "engagement"}'
    
    Result: 
    {
      "strategy": "campaign_strategy_001",
      "status": "created"
    }
    
    Then watch the terminal where API is running!

════════════════════════════════════════════════════════════════
🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════

API won't start?
  → Check .env.local syntax
  → Make sure Python 3.11+ is installed
  → Try: pip install -r requirements.txt --force-reinstall

Telegram auth fails?
  → Check API_ID and API_HASH are correct
  → Make sure phone number format is +COUNTRY_CODE...
  → Wait 24 hours if you just created the API app

Comments not posting?
  → Check target channel has "Comments" enabled
  → Ensure your account isn't in cooldown
  → Check risk scores in dashboard

Port 8000 already in use?
  → Find process: netstat -ano | findstr :8000
  → Kill it: taskkill /PID <PID> /F
  → Or use different port: uvicorn ... --port 8001
