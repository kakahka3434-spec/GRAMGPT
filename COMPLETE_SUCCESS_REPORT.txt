⚡ БЫСТРЫЕ КОМАНДЫ

Проверить здоровье API:
  curl http://127.0.0.1:8000/api/v1/status

Получить список аккаунтов:
  curl http://127.0.0.1:8000/api/v1/accounts | jq

Получить dashboard:
  curl http://127.0.0.1:8000/api/v1/dashboard | jq

Создать кампанию:
  curl -X POST http://127.0.0.1:8000/api/v1/campaigns/create \
    -H "Content-Type: application/json" \
    -d '{"name":"My Campaign","goal":"engagement"}'

Остановить API (в терминале где он запущен):
  Ctrl+C

Перезапустить API:
  cd c:\Users\Administrator\Desktop\ai\GRAMGPT
  uvicorn src.api.main:app --reload --port 8000

Деплоить на VPS:
  bash deploy.sh gramgpt.io your-vps-ip
